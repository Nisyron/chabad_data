Main JSON File Quick Reference

Search Capabilities

Each document includes:

  • Metadata: Hebrew dates, Gregorian dates, author, opening phrases
  • Topics: Hebrew concepts (砖, 爪拽, 转转, etc.)
  • Key Concepts: English translations (Divine Presence, Righteous ones, Lower worlds)
  • References: Biblical, Talmudic, and Chassidic citations
  • Chunked Text: Main text divided into 1,000-1,500 character searchable chunks
  • Footnotes & Glossary: Complete annotations and definitions

Collection Structure

The JSON file contains:

  • collection_metadata - Overview and statistics
  • documents - Array of 195 documents, each with:
    • id - Document identifier (e.g., "1_")
    • reference - Full reference string
    • metadata - Dates, topics, concepts, references
    • content - Main text, chunks, footnotes, glossary
    • search_tags - Keywords for search

Usage Instructions

For Perplexity:

  1. Always use the raw GitHub URL: https://raw.githubusercontent.com/nisyron/chabad_data/main/maamarim_structured.json
  2. Include the URL in your query: "Search in [URL] for..."
  3. Perplexity can search by:
    • Hebrew topics (砖, 爪拽)
    • English concepts (Divine Presence, Righteous ones)
    • Dates (1951, 转砖")
    • References (biblical, talmudic citations)
    • Opening phrases (转 )
    • Document IDs (1_, 1_)
  4. Search fields: The JSON has these searchable fields:
    • metadata.topics - Hebrew topics array
    • metadata.key_concepts - English concepts
    • metadata.opening_phrase - Opening phrases
    • content.main_text_chunks - Searchable text chunks
    • search_tags - All search keywords
